一言不合就装上Linux

发表于 2016-08-23   |     |   阅读次数

作为一个标准的文科生，平时都有听码农们的节目，主要还是以IPN的内核恐慌、IT公论为主，很多期都是反反复复的听，虽然绝大多数听不懂，但还是给了我无数的启发和想学习的动力。

我没有任何语言基础，硬要说的话只会写些简单的HTML+CSS，会用几款建站程序，现在的工作是在一家手机外贸公司做美工，主要工作是做广告图+电商网站页面修改+活动单页。工作内容接触的最多的还是photoshop+FTP+HTML+CSS，算不得前端，因为我连bootstrap用的都不是很熟练，JQ就更别提了，只是个普普通通的美工。内心还是很羡慕会写代码的人，不管是什么语言。

静下心来搭建HEXO的时候，第一次接触命令行，万般不适应，从中出了很多问题，最明显的是shell下输入密码是不显示的，我还想想怎么敲了半天都不显示，难道键盘坏了？虽然经历了不少的挫折，但敲命令的形式搭建好了HEXO成功的那一刻，给了我继续学习的动力，上周日在家捣鼓了半天，把自己的WIN7台式弄成了win7+ubantu双系统，安装过程还是相当顺利的，只是在安装完毕后重启的时候进不去ubantu系统，查了很久是因为显卡的问题，后面跟着一篇教程修改启动文件就莫名其妙的进去了，再次感谢那些在blog上无私奉献自己的心得的人。

进入图形界面的那一刻，心里真的还是很兴奋的。由于自己还是挺想往前端方面发展的，当晚就照着ruby china上的中文手册搭建好了ruby on rails的环境。觉得很神奇的事情是，只需要几个简单的命令环境就搭建完成了，还是很令我这个用windows到现在的小白吃惊，这要是在windows上，估计又得捣鼓很久了。
