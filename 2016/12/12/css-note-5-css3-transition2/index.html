发表于 2016-12-12   |   分类于 笔记 ， css   |  

这个案例是我在网易云课堂上看到的，然后就自己做了一遍。
LINK: Demo

以前我也有见到过类似的，但是根据我自己的经验我是看不出是怎么做出来的，总之觉得很酷，所以这次见到了必须理解了自己做一遍。

原理很简单，用transition和transform相结合，四条线用四个span标签构造。将按钮的文字和四个span标签放入一个节点里，四个span标签相对定位，用top\bottom\left\right，控制初始的位置和hover后的位置。

想要自然的话可以在初始的时候将线条的宽度设置为0，hover再给一个宽度，就会有一个比较舒服的过度。
