css3里transition和transform回顾练习

发表于 2016-12-10   |   分类于 笔记 ， css   |     |   阅读次数

最近工作上一直在忙，忙着各种西方的活动，黑5、圣诞、新年等，虽然平时工作的时候也折腾了比较多的东西，但零零碎碎的没有仔细的记录下来。

以前css3里的transform我不太会用，工作中也经常去别人的站里复制相关的transform效果，所以今天特意自己看看这个效果的用法。

总结出来：
1、相关的兼容要做好
2、不知道为什么，360浏览器对transition和transform支持不太好，基本只能实现半套的动画，剩下的半套直接还原了。百度了下还没有找到解决方法
3、3d功能少用，因为兼容不好

随意做的练习，定义了本地一个比较喜欢的字体（LeviReBrushed）
